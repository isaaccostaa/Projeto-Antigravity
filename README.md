# Landing Page Odontológica

Landing page desenvolvida para uma cirurgiã-dentista, com foco em presença digital, apresentação dos serviços, fortalecimento da credibilidade profissional e conversão de pacientes por meio de contato rápido via WhatsApp.

## Sobre o projeto

Este projeto foi criado como uma landing page moderna e responsiva para o segmento odontológico, com o objetivo de oferecer uma experiência visual profissional, destacar os principais tratamentos da clínica e facilitar o agendamento de consultas.

A proposta do site é unir design elegante, navegação fluida e comunicação estratégica para transmitir confiança ao paciente e valorizar a imagem da profissional no ambiente digital.

## Funcionalidades

- Apresentação da profissional
- Seção de serviços e soluções odontológicas
- Depoimentos e prova social
- Área de transformações / resultados
- Perguntas frequentes
- Seção de agendamento
- Informações de localização
- Botão flutuante para contato via WhatsApp
- Layout responsivo para desktop e mobile
- Navegação com foco em conversão

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS
- Framer Motion
- Lucide React

## Estrutura do projeto

O projeto foi dividido em componentes para facilitar a organização e manutenção da aplicação. Entre as principais seções estão:

- `Header`
- `Hero`
- `AboutDoctor`
- `Solutions`
- `HowItWorks`
- `Testimonials`
- `Transformations`
- `FAQ`
- `Scheduling`
- `Location`
- `Footer`
- `FloatingWhatsApp`

## Objetivo

O principal objetivo deste projeto foi desenvolver uma página de alta conversão para o nicho odontológico, valorizando a apresentação da profissional, melhorando a comunicação dos serviços e oferecendo um canal de contato rápido e acessível para novos pacientes.
